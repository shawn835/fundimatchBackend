# FundiMatch

**FundiMatch** is a web app connecting skilled professionals (“Fundis”) with clients in need of their services...

## Features

- JWT authentication
- Soft-delete user accounts
- Secure login/logout
- Scalable user management
- Resend email service
- Posting jobs
- Editing jobs

## Tech Stack

- Frontend: Vue 3
- Backend: Node.js
- Database: MongoDB
